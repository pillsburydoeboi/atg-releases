# Alt Text Generator

Desktop application for generating accessible alt text for WordPress images using AI.

## Download

See the [Releases](../../releases) page for the latest version.

| Platform | File |
|----------|------|
| Windows | `alt-text-generator-x.x.x-windows.zip` |
| macOS (Intel) | `alt-text-generator-x.x.x-macos-intel.zip` |
| macOS (Apple Silicon) | `alt-text-generator-x.x.x-macos-arm.zip` |
| Linux | `alt-text-generator-x.x.x-linux.tar.gz` |

## Requirements

- Windows 10+, macOS 12+, or Linux
- Valid license file (provided separately)

## Getting Started

1. Download the application for your platform
2. Place your `license.json` file in the same folder as the application
3. Launch the application
4. See **USER_GUIDE.md** (included in each release) for complete setup and usage instructions
